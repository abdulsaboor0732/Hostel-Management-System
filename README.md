# Hostel Management System (C++ Modular OOP)

## Project Overview
An itemized application built natively with C++ utilizing advanced Object-Oriented Programming (OOP) architectures including strict inheritance paradigms, custom static indexing strategies, file manipulation input/output serialization routines, and robust validation structures tracking rooms allocation, student profiles and matching fee structural ledgers cleanly.

## Key Feature Implementations
* **Modular Codebase Separation**: Kept implementation out of primary program control routines by tracking declarations across header components.
* **Inheritance Mapping**: Extends base properties safely using a concrete `Person` paradigm subclassed explicitly into active `Student` properties.
* **Persistence & Input Serializers**: Native storage mechanisms ensuring session continuity across `students.txt` and `rooms.txt`.

## Project Module Structures
* `main.cpp`: Sole structural program entry point routing controls smoothly down secondary component channels.
* `hostel.h`: Central tracking header framework encapsulating structure specifications, state models, and interfaces.
* `hostel.cpp`: Custom built implementation core body layer executing internal validation loops and calculation trees.
